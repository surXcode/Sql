# Qspider SQL Solution
---
This repository contains a complete collection of SQL queries and solutions provided during QSpiders training.  
It includes classroom assignments, practice questions, tests, interview-focused SQL problems, and module-wise exercises.

---

## 📘 **Topics Covered**

- 🔹 Basics of SQL  
- 🔹 DDL (Data Definition Language)  
- 🔹 DML (Data Manipulation Language)  
- 🔹 TCL (Transaction Control Language)  
- 🔹 Joins (Inner, Left, Right, Full)  
- 🔹 Subqueries  
- 🔹 Group By & Aggregate Functions  
- 🔹 String & Date Functions  
- 🔹 Views  
- 🔹 Stored Procedures (if applicable)  
- 🔹 Practice Questions from QSpiders  
- 🔹 Interview-oriented SQL Problems  

---



> *Note: You can organize files as per QSpiders modules.*

## 🚀 **How to Use**

1. Browse the folder related to your SQL module.  
2. Open the `.sql` file to view queries and explanations.  
3. Copy and run queries in your SQL tool (MySQL, Oracle, SQL Server, etc.).  
4. Modify and practice as needed.
